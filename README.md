Built with Laravel in mind

# Base package

* PHP 7.4.5-FPM

# PHP Modules included

* INTL
* PDO
* BCMath
* Sodium
* CType
* JSON
* OpenSSL
* TTokenizer
* XML
* ZIP
* PDO_MySQL (MySQL support)
* PDO_PgSQL (PostgreSQL Support)
* XDebug

# Additional features

* Composer (latest edition) is available in the docker container, with hirak/prestissimo installed
* Git
